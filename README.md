# 🛒 Customer Shopping Behavior Analysis
 
> **Uncovering patterns behind purchasing decisions through data**
> วิเคราะห์พฤติกรรมการซื้อสินค้าของลูกค้าผ่านข้อมูล
 
---
 
## 📌 Overview | ภาพรวมโปรเจค
 
**EN:** This project explores customer shopping behavior using a combination of Python, SQL, and Power BI. The goal is to identify purchasing patterns, segment customers, and generate actionable insights that can support business decision-making in retail.
 
**TH:** โปรเจคนี้วิเคราะห์พฤติกรรมการซื้อสินค้าของลูกค้าโดยใช้ Python, SQL, และ Power BI เพื่อค้นหารูปแบบการซื้อ แบ่งกลุ่มลูกค้า และสร้าง Insight ที่นำไปใช้งานได้จริงสำหรับการตัดสินใจเชิงธุรกิจในภาคค้าปลีก
 
---
 
## 🎯 Objectives | วัตถุประสงค์
 
- **EN:** Analyze customer demographics and purchasing behavior
- **TH:** วิเคราะห์ข้อมูลประชากรและพฤติกรรมการซื้อของลูกค้า
 
- **EN:** Identify top-performing product categories and seasonal trends
- **TH:** ระบุหมวดหมู่สินค้าที่ขายดีและแนวโน้มตามฤดูกาล
 
- **EN:** Segment customers based on purchasing patterns
- **TH:** แบ่งกลุ่มลูกค้าตามรูปแบบการซื้อ
 
- **EN:** Build an interactive dashboard for visual exploration
- **TH:** สร้าง Dashboard แบบโต้ตอบสำหรับการสำรวจข้อมูลเชิงภาพ

## 🚀 Getting Started | วิธีเริ่มต้นใช้งาน
 
### Prerequisites | สิ่งที่ต้องมี
 
```bash
Python >= 3.8
Power BI Desktop (for .pbix file)
```
 
### Installation | การติดตั้ง
 
```bash
# Clone the repository
git clone https://github.com/bI3ear/Customer_Shopping_Behavior_Analysis.git
 
# Navigate to the project directory
cd Customer_Shopping_Behavior_Analysis
 
# Install required libraries
pip install pandas jupyter
```
 
### Run the Notebook | รันโน้ตบุ๊ก
 
```bash
jupyter notebook Customer_Shopping_Behavior_Analysis.ipynb
```
 
### View the Dashboard | ดู Dashboard
 
Open `Customer_behavior_dashboard.pbix` with **Power BI Desktop**
เปิดไฟล์ `Customer_behavior_dashboard.pbix` ด้วย **Power BI Desktop**
